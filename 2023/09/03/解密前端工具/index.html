CSS

CSS就是一个颜料仓库，里面有蓝颜料，红色颜料，等等。

通过[HTML]树枝结点的[class]属性将[HTML]树和很多颜料瓶连接，从而改变树叶，树枝的形态。

JavaScript

[JavaScript]可太像[Scheme]了，不过缺少[Scheme]的宏，倒是一个阉割版的[Scheme]。[JavaScript]和[HTML]通过[input]属性连接。如同一个蚂蚁王国，蚂蚁们能让果子拥有各种各样的味道。
